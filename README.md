# Accounting Frontend

This is the Nuxt 3 frontend application for the Accounting UI.

## Prerequisites

- Node.js (v18+)
- [pnpm](https://pnpm.io/installation) (used for package management)

## Installation Instructions

1. Navigate to the FrontEnd directory:
   ```bash
   cd FrontEnd
   ```
2. Install the necessary node modules using `pnpm`:
   ```bash
   pnpm install
   ```
3. Ensure your environment variables are set up. If there is a `.env.example`, copy it:
   ```bash
   cp .env.example .env
   ```

## Running the Application

To start the development server, run the following command using `pnpm`:

```bash
pnpm dev
```

The application will start locally, typically accessible at `http://localhost:3000`.
